This plugin adds links to specified words in posts and pages.

== Description ==

This plugin adds hyperlinks to specified words in posts and pages. This can be very usefull if you are using complicated words and want to link Wikipedia articles or 
if you just want to improve your seo by linking some interessting words :)

== Installation ==

1. Upload the `aglinker` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Example ==

Your Text:
`Welcome to a blog about google`

Your Plugin-Settings:
`blog|Wordpress software|http://wordpress.com
google|Search engine|http://google.com`

Result:
`Welcome to a <a href="http://wordpress.com" title="Wordpress software" target="_blank">blog</a> about <a href="http://google.com" title="Search engine" target="_blank">google</a>`
